# Teste-A-B-e-Prioriza-o-de-Hip-teses
Aplicação dos frameworks ICE e RICE para priorização de hipóteses de impacto em receita. Condução de teste A/B com análise de conversão, ticket médio e receita acumulada, incluindo identificação de outliers e validação de significância estatística. Projeto resultou em recomendação estratégica.
